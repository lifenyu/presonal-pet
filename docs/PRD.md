# 产品需求文档 (PRD)

## 产品名称
桌宠学习陪伴 (Pet Study Companion)

## 产品概述
一款 Windows 桌面 AI 桌宠应用，以可爱宠物形象常驻桌面，帮助用户管理电脑使用时间、培养健康习惯、提供 AI 学习辅助。

## 目标用户
- 长时间使用电脑的学生和上班族
- 需要提醒休息、护眼、喝水的用户
- 希望追踪自己电脑使用习惯的用户

## 功能规划

### V0 — 桌宠基础 (当前版本)
- [x] 透明无边框窗口，桌宠形象常驻桌面
- [x] 窗口可拖拽、始终置顶
- [x] 系统托盘菜单（显示/隐藏、设置、退出）
- [x] 设置页面入口
- [x] SQLite 数据库初始化

### V1 — 使用监控
- [ ] 监控前台窗口（软件名称、窗口标题）
- [ ] 记录使用时长
- [ ] 空闲状态检测
- [ ] 软件分类（学习/游戏/看剧/聊天/工作/浏览/其他）
- [ ] 使用统计面板

### V2 — 健康提醒
- [ ] 连续使用时间追踪
- [ ] 定时提醒（喝水、休息、运动、护眼）
- [ ] 提醒频率可配置
- [ ] 桌宠动画提示

### V3 — AI 对话
- [ ] OpenAI-compatible API 配置（base_url, api_key, model）
- [ ] 桌宠对话框问答
- [ ] 对话历史记录

### V4 — 进阶功能
- [ ] OCR 截图提问
- [ ] AI 学习总结
- [ ] 语音输入
- [ ] 语音播报
- [ ] 自定义桌宠形象

## 非功能需求
- 内存占用 < 150MB
- CPU 空闲时占用 < 1%
- 启动时间 < 3 秒
- 数据本地存储，不上传用户隐私
